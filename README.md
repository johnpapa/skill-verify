# skill-verify

A Copilot CLI skill that scores and improves AI skill quality.

Combines:
- **Sensei** scoring (frontmatter, triggers, routing clarity)
- **Agent Skills Spec** compliance (structure, clarity, safety)
- **Real-world patterns** from Angular, React, vscode, and 300+ awesome-copilot skills

## Status

🚧 Planning — see [Issue #1](https://github.com/johnpapa/skill-verify/issues/1) for the full plan.

## Install (coming soon)

```
/skills add johnpapa/skill-verify
```

Then say: "verify this skill" or "score my SKILL.md"
